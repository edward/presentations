Thinking like a developer. Part II.

Yeeeehaaaa. It's Prototyping time.

Task:
  • Using an existing website as a base, edit it to create a new site with a different purpose.
Instructions:
  1. Download a code editor unless you already have one. Example: Sublime Text.
  2. Download this file and unzip it
  3. Inside the folder you just unzipped, you'll find two folders: Hyperchat and Durian. Inside each of them is an index.html file. You can open these up in your web browser if you want to see them, and you can open them up in your code editor to edit their HTML.
  4. Pick either Hyperchat or Durian as a base, and edit the html to create your BrandNewWebsite™. You can make a copy of the folder if you don't want to overwrite the original files.
  5. Be as creative as you want. Your app doesn't need to be towards your brief. It could be a silly idea, something you'd like to use, or an idea that would benefit life in Singapore! Up to you.
Bonus:
  • Host your website online. If you have a dropbox account, this is done easily by using the Brace.io app for Dropbox.
  • Post the link to your site to this Google Doc!

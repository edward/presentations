Find all elements in the html with a class of "bamse", and add the class "special" to each one.


Small hint: If you don't want to type '$', you can write 'jQuery' instead.

Here is a css file where the class 'special' is defined

.special {
  border: 2px solid red;
}

Click the buttons below to add/remove the class

$('.bamse').addClass('special');
$('.bamse').removeClass('special');

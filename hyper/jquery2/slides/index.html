    JSON

    JavaScript Object Notation

    The data type that you will love!

    Some types we have seen already

    • Strings 
       var name = "Daniel";
    • Numbers 
       var age = 25;
    • Booleans 
       var isFunny = true;
    • Arrays 
       var fruits = ['apples', 'oranges', 'kiwis'];

    So what is an object?

    
  var daniel = 
    {
      name: "Daniel",
      age: 25,
      gender: "Male",
      city: "Stockholm"
    }

    "A collection of properties"

    Syntax

    var objectName = { 
  propertyName: propertyValue,
  propertyName: propertyValue,
  ...
}

    Another example

    var stockholm = { 
  country: "Sweden",
  population: 871952
}

    Accessing properties

    var daniel = {
  name: "Daniel",
  age: 25,
  gender: "Male",
  city: "Stockholm"
}
    "dot notation" 
    alert(daniel.name);
    alert(daniel.age);

    Changing properties

    daniel.gender = "female";

    Properties can be anything

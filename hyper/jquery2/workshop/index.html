Getting started

  1. Download a copy of this workshop over here. Open the index.html file up in your browser.
  2. Write your answers in the main.js file in the js directory. You ARE allowed to modify the html too...
  3. If your code is not working, the first place to look for problems is in the javascript console in your web inspector.
  4. Google it if stuck. There's a ton of documentation on jQuery
